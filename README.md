# 💰 Personal Finance Dashboard

## Overview

This project is a **Personal Finance Dashboard** built using modern web technologies to help users track their income, expenses, and financial activity in one place.

The application provides a clean and interactive interface for monitoring financial data through **visual charts, transaction tracking, and summary statistics**.

The main goal of this project is to demonstrate **interactive UI development, state management, and data visualization** in a finance-related web application.

---

## Features

* Secure login interface
* Financial overview dashboard
* Total balance summary
* Monthly income tracking
* Monthly expense tracking
* Savings overview
* Expense categorization
* Transaction history table
* Interactive financial charts
* Budget tracking
* Data stored locally in browser
* Responsive dark-themed UI

---

## Technologies Used

**Frontend**

* HTML5 – Application structure
* CSS3 – Styling and layout
* JavaScript – Application logic

**Framework / Libraries**

* React.js – UI components and state management
* Chart.js / Recharts – Financial data visualization
* LocalStorage – Local data persistence

## How to Run the Project

### Run Locally

Clone the repository

```
git clone https://github.com/Dasmohapatraa/personal_finance_dashboard.git
```

Open the project folder.

Install dependencies

```
npm install
```

Run the development server

```
npm run dev
```

Open the application in your browser

```
http://localhost:5174
```

---

### Run Online (GitHub Pages)

If GitHub Pages is enabled for this repository, the project can be accessed directly in the browser:

```
https://Dasmohapatraa.github.io/personal_finance_dashboard/
```
